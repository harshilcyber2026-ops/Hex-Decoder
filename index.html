Background

Hexadecimal (base-16) uses digits 0–9 and letters a–f. Each pair of hex digits represents one byte, which maps directly to an ASCII character.


Example: 0x46 = decimal 70 = ASCII 'F'. No key needed — it is a fixed, reversible encoding.
